* **Базовый рендеринг:** Настроен двойной буфер (DoubleBuffering) для плавной отрисовки на 50 FPS без мерцаний.

# План разработки: Неделя 3 (Выполнено)

## Система кат-сцен, слои и новая физика
* **Локация «Спуск в метро»:** Полностью настроена физика эскалатора (правильный угол наклона, ширина ступеней, отключена возможность прыжка).
* **Слои отрисовки:** Реализовано правильное наслоение графики (физические блоки скрыты -> задний фон -> персонаж -> передний план с перилами). 
* **Оптимизация препятствий:** Улучшены хитбоксы на улице (сугроб разбит на несколько блоков для плавного запрыгивания).
* **Система кат-сцен:** Настроены переходы по триггерам на координатах (переход «Метро -> Вагон») с таймингами и затемнением.

## Ассеты и спрайты
* **Отрисовка главного героя:** Создан детализированный спрайт-лист Яны в пиксель-арте.
* **Разные аутфиты:** Персонаж нарисован в двух вариантах одежды (уличный стиль с курткой/сумкой и домашний в худи).
* **Кадры анимации:** Для каждого аутфита подготовлено по 3 кадра (положение стоя и два уникальных кадра шага с правильным чередованием ног в движении).
