# 𖣂 Станция Лихо (Likho Station) 𖣂

**Жанр:** 2D Stealth-Adventure / Атмосферный хоррор  
**Платформа:** Windows Forms (C# / GDI+)  
**Сеттинг:** Современный славянский пост-хоррор (по мотивам вселенной Нади Совы)

## 📖 Сюжет (Демо-версия 5 минут)

Игрок берет на себя роль **Яны** — студентки художественной академии. После изнурительной ночи за рисованием она прощается с бабушкой и уезжает на учебу в метро. Усталость берет свое: Яна засыпает и просыпается на конечной станции, которая оказывается входом в **Ладный мир**.

Это искаженное, пустынное отражение реальности, где в тумане панельных домов обитают хтонические существа, а за всем наблюдает древнее **Лихо**. Чтобы проснуться и вернуться домой, Яне нужно найти три личные вещи («якоря реальности»), которые выпали из её сумки, и успеть на призрачный поезд.

---

## 🖥️ Ключевые механики

### 1. Платформинг и Физика
Игра представляет собой сайд-скроллер с видом сбоку. Яна может перемещаться по многоуровневым локациям, перепрыгивать препятствия и бездонные провалы. Динамическая камера следит за передвижением игрока по длинным уровням.

### 2. Механика «Задержка дыхания» (Stealth)
В мире игры обитают **Ауки** — духи-невидимки. Их нельзя увидеть обычным взглядом, но можно почувствовать по визуальным помехам и шепоту.
* **Действие:** Удерживание клавиши `С` (или `Ctrl`) заставляет Яну присесть, замереть и задержать дыхание.
* **Риск:** Шкала выносливости (кислорода) быстро тратится. Если она упадет до нуля, персонаж сделает резкий вдох, мгновенно выдав себя врагам в радиусе слышимости.

### 3. Механика «Чутье художника» (Focus Mode)
Яна воспринимает мир через призму своего таланта.
* **Действие:** Клавиша `Shift` обесцвечивает мир, оставляя яркими только «якоря реальности» (квестовые предметы) и силуэты невидимых врагов на короткое время.
* **Назначение:** Позволяет быстро ориентироваться на уровне и быстро находить цели.

### 4. Атмосферная «Серая Хмарь»
Техническая реализация освещения/видимости через GDI+. Из-за густого тумана Изнанки игрок видит мир только в ограниченном радиусе (круге) вокруг персонажа, что создает ощущение клаустрофобии и неизвестности.

## 🕑 План демонстрации

| Время | Этап | Описание |
| :--- | :--- | :--- |
| **0:00 - 1:00** | **Реальность** | Пролог на кухне с бабушкой. Обучение управлению и взаимодействию (взять сумку). |
| **1:00 - 1:30** | **Переход** | Сцена в вагоне метро. Засыпание и пробуждение в Ладном мире. |
| **1:30 - 4:15** | **Изнанка** | Исследование поверхности. Стелс-сегмент с Ауками, поиск 3-х предметов с помощью Чутья. |
| **4:15 - 5:00** | **Финал** | Побег на платформу. Возвращение в реальный мир (пробуждение). |

## 🕹️ Управление
* **AD / Стрелки** — Перемещение персонажа.
* **Space (Пробел)** — Прыжок.
* **C / Ctrl (Удержание)** — Присесть / Задержать дыхание (скрытность).
* **Shift (Удержание)** — Чутье художника (подсветка).
* **E** — Взаимодействие (подбор предметов, чтение записок, переход в двери).
